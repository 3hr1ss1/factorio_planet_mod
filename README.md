# factorio_planet_mod
A factorio mod for our Software development Subject

## contents

### Flora and Fauna
- New planet (desart)
- New mobs and spawners
- New Ressources (sand, glass)

### Items and Crafting
- Short handed Inserter (Puts Items on the front row of the belt, instead of the back one)
- Solar oven (Crafting Speed 4, Base Productivity 50%)
